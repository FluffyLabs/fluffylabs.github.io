# Fluffy Labs Website

This repository is now a Vite + React app and uses `@fluffylabs/shared-ui` directly for layout primitives.

## Stack

- Vite
- React
- `@fluffylabs/shared-ui`

## Shared UI usage

The website uses:

- `Header`
- `AppsSidebar`
- `Content`

Imported from `@fluffylabs/shared-ui` in `src/App.jsx`.

## Development

1. `npm ci`
2. `npm run dev`

## Build

- `npm run build`
- `npm run preview`
